Instructions:

1) Python must be installed on the computer to run this program.

2) Run the prototype_database.py file.

	Example:

	In Windows, if the files are on the desktop, open a command prompt and type:

	py desktop\prototype_database.py


3) Run the prototype_interface.py file.

	Example:

	In Windows, if the files are on the desktop, open a command prompt and type:

	py desktop\prototype_interface.py


4) Press any button on the left side of the screen to open a popup window.  Fill in the labeled fields with data, then press the enter button.  The results will be displayed in the output box on the main window.  Scroll up and down to view current and previous output results.

5) To close the program, just close the window.
